**Nom** : RANDRIAMIALY JOHNNE Koloina Jacquie
**Matricule** : 352/L1A/25-26
**Classe** : L1A
**Sujet choisi** : Création d'une application permettant d'ajouter, modifier et 
supprimer une tâche, définir une propriété, changer le statut et affecter une 
tâche à un utilisateur en Symfony

## Description

Application web développée avec Symfony permettant la gestion complète de tâches :
- Ajout, modification et suppression de tâches
- Définition de propriétés (priorité, description, échéance)
- Changement de statut (à faire / en cours / terminée)
- Affectation d'une tâche à un utilisateur
- Authentification (inscription / connexion)
- Gestion des rôles (utilisateur / administrateur)

## Technologies utilisées

- PHP 8 / Symfony 7
- Doctrine ORM
- MySQL (base de données)
- Bootstrap 5 (interface)
- Twig (templates)

## Fonctionnalités principales

- CRUD complet des tâches
- Système d'authentification sécurisé
- Filtrage "Mes tâches" vs "Toutes les tâches"
- Tri automatique par priorité
- Interface responsive et moderne
